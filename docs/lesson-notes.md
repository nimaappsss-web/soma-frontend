# Lesson Notes — Teacher Experience

## 1. Auto-Growing Textareas
Each textarea expands as the teacher types, up to full viewport height. No scrolling within the field — the content determines the height.

## 2. Single-Tap Field Navigation
Tap any section header or its preview — cursor jumps directly to that field. Eliminates scroll-target-tap corrections.

## 3. Text Expansion / Macros
Short codes auto-expand into full phrases:
- `@@so` → "Students will observe that…"
- `@@eval` → "Evaluation: "
- Ships with a library of common Nigerian lesson note phrases.
- Teachers can add custom macros.

## 4. Keep Fields Shallow
Many small fields (Topic, Objective, Step 1, Step 2, Evaluation, etc.) instead of one giant textarea. Each field is a short burst of typing — phones handle short bursts well.

## 5. Save Draft + Resume
One tap saves everything locally. Teacher closes the app and returns later to pick up exactly where they left off. No lost work.

## 6. Offline-First
Everything works without internet. Typing is local, syncs in the background when connected. No "I'll do it later" avoidance.

## 7. Smart Defaults from Previous
Auto-fills the next lesson note with last week's structure — same topic format, evaluation style, step layout. Only the content that's actually different needs editing. Cuts typing by 60–70%.
